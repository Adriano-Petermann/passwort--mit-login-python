# passwort--mit-login-python
Einfacher Passwort-Checker mit Login in Python 

# Funktionen:
Überprüfung auf Großbuchstaben (nicht erlaubt)
Mindestlänge von 6 Zeichen
Keine Sonderzeichen erlaubt
Passwort muss mindestens eine Zahl enthalten
Speicherung des Passworts in einer Datei
Einfacher Login durch Vergleich des gespeicherten Passworts
# Nutzung:
Programm starten
Passwort eingeben
Falls gültig:
Passwort wird gespeichert
Login kann getestet werden

( Das Passwort wird im Klartext gespeichert. )
( In echten Anwendungen würde man Passwörter verschlüsseln. )
